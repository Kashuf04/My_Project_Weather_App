# Weather App

This is a simple Android Weather Application developed using Android Studio and Java.

## Features

* Search weather by city name
* Displays:

  * Temperature
  * Weather condition
  * Humidity
  * Wind speed
* Uses OpenWeatherMap API
* Simple and user-friendly interface

## Technologies Used

* Android Studio
* Java
* XML
* Volley Library
* OpenWeatherMap API

## How the App Works

The user enters a city name and clicks the "Get Weather" button. The application sends a request to the weather API and displays live weather information on the screen.
